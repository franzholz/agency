sys_language_uid
-----------------

..  confval:: sys_language_uid
    :name: sys-language-uid
    :type: integer

    The system language uid as in config.sys_language_uid. You can override this on the extension 
    level for each table and field to use Dmail categories in multiple languages under all website 
    languages. Do not set it if you want to use the settings from  config.sys_language_uid in the 
    case when the Dmail categories are translated.
