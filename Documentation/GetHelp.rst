=================
Where to get help
=================

You can get help in the `TYPO3 Slack chat <https://typo3.org/community/meet/chat-slack>`_.
Ask a question in the `official TYPO3 forum <https://talk.typo3.org/c/typo3-questions/19>`_ 
or in the `Jambage.com Forum <http://www.jambage.com/kontakt/forum.html>`_
or contact the extension author via e-mail.

..  _report-issues:

Report Issues
=============

You can report issues at `Agency Issues (Github) <https://github.com/franzholz/agency/issues>`_.
